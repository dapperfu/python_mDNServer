#mDNServer
Mulitcast Domain Name Server because Windows.

Upstream mDNS lookup server for devices that don't have mDNS.

## Motivation

1. mDNS is awesome.
1. Not all devices or OSs can use mDNS. (Looking at you Microsoft).
